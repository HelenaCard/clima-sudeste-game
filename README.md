## Financiamento Sustentável e Climático para as Comunidades do Sudeste - Desafio RPNSAR & Plataforma EnergyC

### Objetivo: 

Simular de forma dinâmica e didática a gestão de recursos e investimentos na prática sobre financiamento sustentável e climático no sudeste do Brasil.

### Descrição

O projeto facilita o entendimento sobre os investimentos das finanças sustentáveis a partir dos materiais que foram produzidos durante o DesAFIO RPNSAR. Assim, todos as informações forma retiradas de locais públicos
e de pesquisas realizadas nos materias (Ebook, cartilha e detalhamento).


### Jornada do game
Ao entrar na tela inicial encontrará dois botões um  para ir até o game e outro para saber mais sobre a temática do DESAFIO.
